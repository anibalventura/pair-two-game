Easy to play memory game with emojis, build with SwiftUI and MVVM pattern.

## Features

- CRUD a Emoji set.
- Keep score of the game.
- Reset current game.
- Animations.
- Splash screen.
- Dark mode support.
- Localization support (English/Spanish).

## Development

- iOS Development Target **15**
- SwiftUI **3.0**
- Swift **5**
- Xcode **13.2**

## Build

Clone the repository and run!
